## Key Design Elements

### Typography
- **Section headers**: 1.5rem, bold, dark gray (#1f2937)
- **Member names**: 1.1rem, bold, dark gray (#1f2937)
- **Email**: 0.9rem, medium gray (#6b7280)
- **Details**: 0.9rem, standard gray (#374151)

### Spacing
- **Card padding**: 20px all sides
- **Card gap**: 15px between cards
- **Section gap**: 30px between columns
- **Detail rows**: 10px gap between each

### Interactive Elements
- **Buttons**: Gradient backgrounds, hover lift effect
- **Cards**: Hover shadow and border color change
- **Scrollbars**: Teal color matching site theme

## Icons Used (SVG)

All icons are stroke-based SVG with no fill:

- **Phone**: Classic phone handset icon
- **Clock**: Circle with clock hands
- **User**: Person silhouette
- **Arrow**: Back arrow for navigation button

No emoji or text symbols used anywhere.
